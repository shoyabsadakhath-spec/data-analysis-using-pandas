# Data Analysis Web Application

A complete tool to upload CSV files, clean data, generate statistics, insights, and interactive visualizations.

## Features

- Upload CSV (up to 50MB)
- Data cleaning: remove duplicates, handle missing values (mean/median/mode)
- Automatic insights (missing data, skewness, outliers, correlations)
- Interactive charts (histogram, bar, scatter, correlation heatmap)
- Export cleaned CSV, statistics as JSON
- Standalone CLI script and demo script included
